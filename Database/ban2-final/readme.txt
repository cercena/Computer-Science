Projeto feito em Java via IDE NetBeans, para replicar, basta abrir um projeto passando a pasta como parâmetro e executar o arquivo Principal.
O arquivo .jar do JDBC está dentro da pasta e propiamente vinculado com o projeto, caso não esteja, irá aparecer um aviso para vincular ao abrir o projeto.
O arquivo .sql de backup do banco foi gerado a partir do pgAdmin 4 versão 4.30.
